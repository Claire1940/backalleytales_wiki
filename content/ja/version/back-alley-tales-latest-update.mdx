「完璧な結末」に到達するには、細心の注意を払った探索が必要です。最終幕で必要な対話オプションをアンロックするには、最初の3つのチャプターですべての環境の手がかりを見つける必要があります。2026年のアップデートでは、ポーズメニューに「手がかりトラッカー」を追加することで、これらの要件がより明確になりました。

| エンディング名 | 条件 | 結果 |
| :--- | :--- | :--- |
| **真実** | 15個の手がかりをすべて発見、情報屋の高い信頼 | 物語の完全な解明 |
| **脱出** | 最小限の手がかり、低い対立 | 主人公が街を去る |
| **沈黙** | タイム制限対話の失敗、手がかりの見落とし | 未解決の謎 |
| **裏切り** | 敵対者との高い交流 | 最も暗い結末 |

「裏切り」エンディングは必ずしも失敗ではありません。それは「真実」エンディングでは得られない、都市の伝承に関する独自の視点を提供します。コンプリートを目指すプレイヤーは、両方を見ることをお勧めします。
