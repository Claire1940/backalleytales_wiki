<Section icon={Heart} title="主要NPCと同行者のダイナミクス">

バックアレイテイルズにおけるNPCは、進行の主な源です。多くのキャラクターは、特定の章であなたを助けてくれる同行者として行動しますが、他のキャラクターは特定の場所に留まり、サービスや伝承を提供します。関係性システムはこれらのキャラクターとの親密度を追跡し、通常のプレイでは利用できないユニークな「キャラクターエンディング」を解放することができます。

| NPCタイプ | 交流スタイル | 一般的な報酬 |
| :--- | :--- | :--- |
| **情報屋** | 取引ベース、秘密重視 | マップデータ、パスワードの手がかり |
| **生存者** | 保護的、共感ベース | セーフハウスへのアクセス、アイテム |
| **執行者** | 挑戦的、戦闘ベース | 縄張りへのアクセス、装備 |
| **商人** | 経済的、貿易ベース | 消耗品、アップグレード |

<Section icon={Target} title="キャラクターの関係性とエンディングへの影響">

バックアレイテイルズにおいて、「全エンディング」達成の目標は、キャラクターとの関係をいかに管理するかに大きく依存します。ゲームは、すべての主要キャストメンバーにわたるあなたの「道徳的整合性」と「信頼スコア」を追跡します。これらの目に見えないステータスは最終章で最高潮に達し、どのキャラクターがあなたの味方をし、どのキャラクターが敵対するかを決定します。

| エンディング影響要因 | 説明 | 結末への影響 |
| :--- | :--- | :--- |
| **信頼スコア** | すべての肯定的交流の合計 | 誰がフィナーレを生き残るかを決定 |
| **共有された証拠** | NPCに明かされた真実の量 | エンディングの物語のトーンを変化させる |
| **裏切り回数** | 味方よりも自己利益を選んだ回数 | 「孤独」または「裏切り者」エンディングを誘発 |
| **秘密の発見** | NPCの隠された過去を見つけること | 「真の」キャラクターエンディングを解放 |

<Callout type="tip" title="複数のセーブスロット">
2026年にすべてのキャラクター関連のエンディングを体験するには、ゲームの中盤にある主要な「決定ポイント」の前に複数のセーブファイルを保持しておくことをお勧めします。
</Callout>

キャラクターシステムの美しさは、その予測不可能性にあります。序盤で無視したキャラクターが、終盤の危機において唯一の希望になることもあり、キャスト全員と関わりを持つプレイヤーには報酬が与えられます。

</Section>
