<Section icon={Map} title="核心的なゲームプレイメカニズムとシステム">

基本的なシステムを理解することは、**『バックアレイテイルズ』のチュートリアル**における最初のステップです。本作は、観察と直接的なインタラクションのユニークな融合によって成り立っています。街の暗い路地を監視する警備員として、プレイヤーの主なインターフェースはCCTV（防犯カメラ）システムになります。これにより、一般の市民が決して目にすることのない出来事を目撃することができます。この視点は、ストーリーを進めるために必要な証拠を集める上で極めて重要です。

| システム | 主な機能 | 主な操作 |
| :--- | :--- | :--- |
| **CCTVモニター** | 観察 | カメラ映像を切り替えてイベントを探す |
| **現地調査** | 直接行動 | カメラで発見した場所を訪れる |
| **会話ツリー** | ストーリー分岐 | 返答を選択してNPCの信頼度に影響を与える |
| **手がかり手帳** | 進行状況の追跡 | 集めた証拠や目的を確認する |

インタラクションシステムは状況に応じて変化します。環境内の「ホットスポット」に近づくとカーソルが変化し、ゴミ箱を調べる、目撃者に話しかけるなどのアクションが可能であることを示します。ゲーム内の昼夜サイクルにおける特定の時間帯にしか発生しないイベントもあるため、効率的な時間管理が極めて重要です。

<Callout type="tip" title="観察が鍵">
環境のわずかな変化を見逃さないよう、常にカメラ映像を監視しましょう。午後10時には施錠されていたドアが、午前2時には半開きになっていることがあり、そこから新しい調査ルートが開かれる場合があります。
</Callout>

<Section icon={Target} title="ステップバイステップ攻略：調査の開始">

ゲームの序盤を進めるには、順を追ったアプローチが必要です。この**『バックアレイテイルズ』のチュートリアル**では、後で進行が滞る原因となるような重要なストーリートリガーを見逃さないよう、初期のシーケンスを詳しく解説します。

<Steps>
  <Step num="1" title="警備室へアクセス">
    まずは警備の拠点に慣れることから始めましょう。メインモニターを操作して、4つの主要なカメラアングルを表示します。これらを切り替えて、最初のイベントが発生する「アクティブな」シーンを特定してください。
  </Step>
  <Step num="2" title="最初の異変を特定">
    3番目のカメラ映像（狭い路地）で、特定のNPCの動きに注目してください。イベントが終了したら警備室を出て、実際の場所に向かい、落とし物を見つけましょう。
  </Step>
  <Step num="3" title="NPCとの会話を開始">
    通りに戻り、ネオンサインの近くにいる目撃者を見つけます。会話の選択肢から「尋問」を使用してください。ここでの成功は、ステップ2でアイテムを発見しているかどうかにかかっています。
  </Step>
  <Step num="4" title="第2地区の解放">
    目撃者の報告を完了すると、北地区へのゲートが解放されます。このエリアには、より複雑なパズルや、より重要なキャラクター同士のやり取りが含まれています。
  </Step>
</Steps>

<Callout type="warning" title="取り返しのつかない選択">
ステップ3の進行中は注意してください。一部の会話の選択肢はNPCの好感度を永久に下げてしまう可能性があり、その周回プレイ中にそのキャラクター固有のエンディングに到達できなくなる場合があります。
</Callout>

| マイルストーン | 条件 | 報酬 / 解放要素 |
| :--- | :--- | :--- |
| **最初の手がかり** | 路地のゴミを調査する | 目撃者との会話が可能に |
| **地区へのアクセス** | 最初の報告を完了する | 新しいマップ位置とNPC |
| **CCTVのアップグレード** | 証拠を5つ集める | ズーム機能と暗視機能 |

<Section icon={Trophy} title="エンディングガイド：条件と結果">

ゲームは、キャンペーン中のプレイヤーの行動や選択に基づいて、複数の結末が用意されています。この**『バックアレイテイルズ』のチュートリアル**では、主なエンディングカテゴリーの条件をまとめています。

| エンディングの種類 | ストーリーへの影響 | 主な条件 |
| :--- | :--- | :--- |
| **通常エンディング** | 中立的な解決 | 基本的な証拠のみでストーリーを完了する |
| **真エンディング** | 完全な真相解明 | すべての主要な手がかりを発見し、NPCの好感度を高く保つ |
| **バッドエンディング** | 未解決の謎 | 主要なNPCの保護に失敗する、または期限を逃す |
| **シークレットエンディング** | 超自然的な展開 | すべての秘密のアイテムと隠されたVHSテープを回収する |

「真エンディング」を達成するには、時間管理と徹底的な調査のバランスが取れた完璧なプレイが必要です。通常、時間制限のあるインタラクションシーケンスを伴う各キャラクターの「クライシスイベント（危機イベント）」で、全員を確実に生存させなければなりません。

<Callout type="tip" title="セーブデータの管理">
新しい「地区の解放」が始まるたびに、複数のセーブファイルを作成しておきましょう。これにより、特定のエンディングの条件を逃したことに気づいた場合でも、後からやり直すことができます。
</Callout>
