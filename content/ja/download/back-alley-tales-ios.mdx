Appleデバイスにサードパーティ製アプリケーションをインストールするには、Androidシステムとは異なるアプローチが必要です。iOSはサンドボックス環境内で動作するため、App Storeで直接配信されていないゲームを実行するには、構成プロファイルや代替アプリ配信方法を利用する必要があります。このセットアッププロセスにより、システムの整合性を維持しながら、ゲームファイルを正しく実行できます。

インストールを開始する前に、インターネット接続が安定していることを確認してください。接続が不安定な場合、ダウンロード中に構成プロファイルが破損し、最初からプロセスをやり直す必要が生じることがあります。

<Section icon={Shield} title="システム要件＆互換性">

モバイルハードウェアでグラフィック負荷の高いインタラクティブなゲームを実行するには、十分な処理能力とメモリ割り当てが必要です。iOSデバイスはRAMの管理が厳格なため、システムメモリが不足するとバックグラウンドプロセスがゲームを強制終了してしまいます。

シーン切り替え時の頻繁なクラッシュを防ぎ、スムーズな体験を確保するために、以下のハードウェア仕様を確認してください。古いデバイスではサーマルスロットリング（過熱を防ぐためにプロセッサの動作速度を低下させる機能）が発生し、フレームレートの低下（カクつき）を招くことがあります。

<Callout type="info" title="推奨ストレージ容量">
デバイスに少なくとも **5GBの空き容量** を確保してください。実行中にゲームアセットを展開するため、システムが一時的なストレージ領域を必要とします。
</Callout>

<Section icon={Target} title="ステップバイステップ設定ガイド">

デバイス上でゲームクライアントを正常に設定するには、以下の手順に従ってください。これらの手順をスキップすると、アプリケーションを起動しようとした際に「信頼されていないエンタープライズデベロッパ」エラーが発生する可能性があります。

<Steps>
  <Step num="1" title="プロファイルのダウンロード">
    iOSデバイスでSafariブラウザを開き、検証済みのダウンロードポータルにアクセスします。ダウンロードリンクをタップして構成プロファイルを取得します。システムプロンプトが表示され、プロファイルのダウンロード許可を求められます。「**許可**」をタップして進めます。
  </Step>
  <Step num="2" title="システム設定を開く">
    ダウンロードが完了したら、ブラウザを閉じてiOSの標準「**設定**」アプリを開きます。設定メニューの最上部に「**プロファイルがダウンロードされました**」という新しい通知バナーが表示されているはずです。このバナーを直接タップします。
  </Step>
  <Step num="3" title="プロファイルのインストール">
    プロファイルのインストール画面で、右上の「**インストール**」をタップします。プロンプトが表示されたら、デバイスのパスコードを入力します。承諾画面を確認し、画面下部の「**インストール**」をもう一度タップして確定します。
  </Step>
  <Step num="4" title="デベロッパ証明書を信頼する">
    プロファイルにエンタープライズデベロッパの信頼が必要な場合は、「**設定**」 > 「**一般**」 > 「**VPNとデバイス管理**」に移動します。「エンタープライズApp」セクションで、プロファイルに関連付けられているデベロッパ名を選択し、「**信頼**」をタップします。
  </Step>
</Steps>

<Callout type="success" title="検証完了">
信頼されると、デベロッパのラベルのステータスが「信頼済み」に変わります。これでホーム画面に戻り、ゲームのアイコンをタップして起動できます。
</Callout>

インストールプロセス中に、システムアラートやインストールのブロックに遭遇することがあります。これらの問題を迅速に解決するには、以下のトラブルシューティング対応表を参照してください。

| 問題 | 考えられる原因 | 推奨される解決策 |
| :--- | :--- | :--- |
| **プロファイルがダウンロードできない** | Safariのブロックまたは接続不良 | Safariのキャッシュをクリアし、Wi-Fi環境で再試行する |
| **プロファイルのバナーが表示されない** | ダウンロードの登録失敗 | 「一般」 > 「VPNとデバイス管理」に手動で移動する |
| **検証失敗エラー** | Appleによる証明書の取り消し | プロファイルを削除し、更新されたバージョンをダウンロードする |
| **アプリが起動直後にクラッシュする** | システムRAMの不足 | デバイスを再起動し、すべてのバックグラウンドアプリを閉じる |

</Section>

<Section icon={Swords} title="ゲームプレイ要素＆コツ">

このゲームは、プレイヤーが防犯カメラを監視し、周囲を分析し、暗い路地裏で発生する物語の出来事をつなぎ合わせていく探偵シミュレーションとして機能します。攻略には、鋭い観察眼、タイミング、そして捜査リソースを効果的に管理することが不可欠です。

プレイヤーは異なるカメラ映像を切り替え、タイムスタンプを調整して過去の出来事を確認し、画面上で強調表示されているオブジェクトを調べる必要があります。すべてのストーリー分岐を明らかにするには、アクションポイントや時間制限の管理が重要になります。

<Callout type="tip" title="観察のコツ">
背景の細部に細心の注意を払ってください。多くの重要なアイテムや手がかりは、夜間サイクルの特定の時間帯にのみ出現します。タイムスクロールツールを使用して、シーンを1コマずつスキャンしましょう。
</Callout>

さまざまなアクションが進行にどのように影響するかを理解することが、すべてのエンディングを解放する鍵となります。以下の表は、主なインタラクション要素とその結果をまとめたものです。

| アクション | トリガー条件 | 期待される結果 |
| :--- | :--- | :--- |
| **カメラの切り替え** | UI上のカメラアイコンをタップ | ライブ映像を別の場所に切り替える |
| **時間の巻き戻し** | 画面下部のスライダーをドラッグ | シーンを夜の早い時間帯に巻き戻す |
| **手がかりの調査** | 画面内で点滅するオブジェクトをタップ | 証拠ログにアイテムを追加し、会話を解放する |
| **キャラクターとの対話** | イベント中にNPCを選択 | 選択肢による分岐ルートを発生させる |
| **シーンのズーム** | 画面をピンチアウト（ズームイン） | 暗い隅に隠された詳細を明らかにする |
